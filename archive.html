Proceedings

Proceedings PDF and/or presentation files can be shared with the CS23 Zenodo community page. The deadline for uploading is August 31. Detailed submission instructions of the Zenodo page are available here (PDF). It is basically the same as the eposter upload steps, but please select the appropriate one in the “Resource type” selection (see Step 05 of the above instruction PDF).

For a written conference proceeding, PDF format with reasonable number of pages (e.g., 4-8 pages for contributed talks and 10-15 pages for invited talks) will be strongly recommended. We have prepared the CS23 LaTeX template in Overleaf, so you are encouraged to use this by simply copying this in your Overleaf (or downloading the template files from Overleaf to compile in your own LaTeX environment).

The LOC team will have a quick format (e.g., meta data) review after it is uploaded so please keep an eye on your e-mails in case we contact you for revision of the format errors. All submissions are assigned a Digital Object Identifier (DOI) number by Zenodo, making contributions citable. Please do not forget to include the correct title, author list, and abstract in the metadata (see STEP 05 of the instruction).

Submission to the arXiv in parallel with submission to the conference proceedings is encouraged. However, if you submit to the arXiv, you must make sure to update the metadata in your Zenodo collection properly so that it does not get listed twice in ADS.
If you have questions in the submission process, please reach out to the organizing team.
